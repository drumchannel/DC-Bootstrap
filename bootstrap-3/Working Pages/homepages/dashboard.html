Rudiments on the Drum Set

If you haven't already gotten familiar with your rudiments with the Rudimental Crash Course master class you should start there! In this master class, we will cover some ways you can begin to apply the different rudiments within grooves and fills. Some are obvious like the single stroke roll, but you will be amazed at what you can do with paradiddles and rolls! Click below to get started.
